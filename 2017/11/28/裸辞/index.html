“裸辞”三次，我得到了什么？

发表于 2017-11-28 | 分类于 个人随笔

#什么是裸辞？

“裸辞”这个词，没有它的前辈“裸奔”、“裸婚”有名。“裸”字有种奋不顾身、任性而为的意味。“辞”，炒鱿鱼的意思，与工作挂钩。“裸辞”是指职场人员在没有找好下家，就不顾一切离职。

#为什么裸辞？

裸辞的理由可以有很多，有句话说的好，“没有哪份工作是没有委屈的”，人都会给自己的行为找到适当的理由，所以这里我只说我三次裸辞的理由。第一家：工资太低，第二家：公司效益不好，第三家：待遇不公平。简单明了的理由，可是人生如果有这么简单就好了。可能大家会问：看你的理由都是公司的问题，你自己的问题都没有嘛。我想说肯定有，而且是占大部分，下面我分析分析自己的原因。

##1. 完全没有个人职业规划。

##记得2010年，小镇上的网吧，两个学生模样的人坐在电脑前，屏幕上面显示高考后大学志愿填写：第一行：计算机信息安全管理，旁边座位的同伴说：“现在物流管理挺火的，你可以报那个试试”，所以第一行：物流管理，第二行：计算机信息安全管理。我这7年来的人生轨迹不知是否是那个时候的决定改变的。后来，学校有职业规划课，现在记不得怎么糊弄过去的。那个时候真不懂，你自己的人生是糊弄不过去的！

##2.当下生活的负担小

##“一人吃饱，全家不饿”的状态，让我这万年单身狗有种优越感，以为自己是单身贵族，可是我知道我这优越感是畸形。别人都超我人生路途两圈，我却只是关注他那沉重的脚步。生活的负担小，并不意味这我对钱的欲望低，真实是完全相反，我现在对钱很焦虑，因为我知道父母在一天天老去，我的实力赶不上他们老去的速度。所以我只能说“当下”的负担给了我裸辞的勇气。看到公司老员工的状况，我不愿意浪费青春，听到社会月入的字眼，我感觉心被提起摔下。

##3.对自己的状态不满意

##每当自己无所事事坐在电脑前，每当自己工作重复一遍又一遍，每当自己心累却无处诉说，我脑中就蹦出两个词，紧接着就幻想该过的生活，如同寒暑假幻想自己该怎么过一样。裸辞这个词变成一个按钮，刺激着大脑神经皮层，如同上瘾一般，最后做出选择，解放自己。

#裸辞之后怎么办？

##这是我三次裸辞的经验之谈。裸辞之后就是空窗期，根据个人不同，空窗期的时间也不同。我两次空窗期都达到5-6月份。心理状态依次：解放—享受—颓废—振作—失望—享受—颓废—-振作—失望—焦虑，来回往复。那时候的你是自闭，极度不自信，消极，易碎。在一个月不曾出门，不曾说话，我躺在床上，棉被好重，感觉时间和命运都压在上面。

在此我劝戒大家：裸辞可以，但一定要规划好空窗期，并严格执行！

##1.必须要列好时间表

##你可以不工作，但是时间不能不列表。因为人都是懒惰的，更是在无人监管的状态下，你更容易让自己滑倒，并不愿意站起来。每天浑浑噩噩，人生靠明日，万事蹉跎。计划中一定不能外出，因为只有强迫自己出去，才能调整好每天的心态，掌控自己。在这段空窗期，你可以充电，做自己想做但是因工作没有做的事。

##2.尝试裸辞时候的想法

##空窗期，你可以尝试自己裸辞的想法。这样失败，你也可以知道自己到底能坚持到哪一步，喜欢哪一条路。不然你会再次陷入到裸辞的恶性循环中。

##3.找理由去陪陪家人

##裸辞，你可以不告诉家人，但是在空窗期，请你抽出时间、找出理由去陪陪家人。因为裸辞的你需要家人给予的力量，因为父母不会怀疑你拙劣的理由。家，永远是你遮风挡雨的港湾。永远有期待你回家的眼神。更永远会有你奋斗的源泉！
